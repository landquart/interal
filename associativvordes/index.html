5) Локальная база частотных рангов

Формат: язык → { слово: ранг }. Можно сгенерировать из wordfreq/top_n_list.

Как расширить инструмент

Для качественной автоматизации лучше заранее подготовить JSON по каждому языку: слова из Kaikki/Wiktionary + частотные ранги из wordfreq.
